Comparing distribution-shift interventions for a two-drug HIV cohort using LMTP

Estimated reading time: 35 minutes

Learning objectives
  • Distinguish four types of adherence distribution shifts and the causal questions each one answers
  • Implement marginal quantile mapping, conditional quantile mapping, mean-calibrated additive shifts, and fixed additive shifts for lmtp
  • Compare shift estimates and explain why agreement or disagreement between approaches is informative
  • Connect shift interventions to the forgiveness curve and explain how each approach summarizes the counterfactual risk surface
  • Articulate the mediation decomposition of drug effects into adherence-mediated and pharmacological components

20 Overview

Chapter 3.3 introduced a distribution-shift intervention using quantile mapping to compare two antiretroviral drugs under a common adherence distribution. That approach is one principled way to operationalize the question “what if Drug B patients adhered like Drug A patients?” But it is not the only way. Different shift functions encode different assumptions about what “adhering like Drug A” means, and they answer subtly different causal questions.

This supplement implements four approaches on the same simulated data:

  1. Marginal quantile mapping — map Drug B adherence to the Drug A distribution via the empirical CDF.
  2. Conditional distribution mapping — same idea, but condition on baseline covariates so the mapping respects patient characteristics.
  3. Mean-calibrated additive shift — add a constant to Drug B adherence so its mean matches Drug A’s.
  4. Fixed additive shift — increase every patient’s adherence by a small fixed amount, regardless of drug.

Each approach defines a different modified treatment policy for lmtp. We run all four, compare estimates, and explain what scientific question each one answers.

21 Simulated Data

We reproduce the two-drug HIV cohort from Chapter 3.3.

Show code 
library(tidyverse)
set.seed(2026)
n <- 2000
T_max <- 4

# --- Baseline ---
W_age    <- rnorm(n, 38, 10)
W_male   <- rbinom(n, 1, 0.6)
W_cd4_bl <- pmax(50, rnorm(n, 400, 130))
W_vl_bl  <- rnorm(n, 4.0, 0.6)
Drug     <- rbinom(n, 1, 0.5)

# --- Containers ---
L_cd4  <- matrix(NA, n, T_max)
L_vl   <- matrix(NA, n, T_max)
A_adh  <- matrix(NA, n, T_max)
C_cens <- matrix(0,  n, T_max)
alive  <- rep(TRUE, n)

for (t in 1:T_max) {
  if (t == 1) {
    cd4_prev <- W_cd4_bl
    vl_prev  <- W_vl_bl
    adh_prev <- ifelse(Drug == 1, rbeta(n, 8, 2), rbeta(n, 5, 3))
  } else {
    cd4_prev <- L_cd4[, t - 1]
    vl_prev  <- L_vl[, t - 1]
    adh_prev <- A_adh[, t - 1]
    cd4_prev[is.na(cd4_prev)] <- W_cd4_bl[is.na(cd4_prev)]
    vl_prev[is.na(vl_prev)]   <- W_vl_bl[is.na(vl_prev)]
    adh_prev[is.na(adh_prev)] <- 0.5
  }

  L_cd4[, t] <- pmax(50,
    cd4_prev + rnorm(n, mean = 40 * adh_prev + 10 * Drug - 15 * (1 - adh_prev), sd = 35) -
    0.3 * (W_age - 38)
  )

  L_vl[, t] <- pmax(0,
    vl_prev + rnorm(n, mean = -1.5 * adh_prev - 0.2 * Drug + 0.3 * (1 - adh_prev), sd = 0.3)
  )

  lp_adh <- 0.8 * Drug + 0.3 * adh_prev +
    0.001 * (L_cd4[, t] - 400) - 0.2 * L_vl[, t] -
    0.005 * (W_age - 38) + 0.1 * W_male + rnorm(n, 0, 0.3)
  A_adh[, t] <- plogis(lp_adh)

  # Censoring indicator: lmtp expects 1 = uncensored (still observed), 0 = censored
  lp_cens <- -4.5 - 0.002 * L_cd4[, t] + 0.4 * L_vl[, t] -
    0.8 * A_adh[, t] + 0.3 * (1 - Drug)
  censored_now <- rbinom(n, 1, plogis(lp_cens)) * alive
  C_cens[, t] <- 1 - censored_now  # flip: 1=observed, 0=censored
  alive <- alive & (censored_now == 0)

  if (t < T_max) {
    idx_gone <- !alive
    A_adh[idx_gone, (t + 1):T_max]  <- NA
    L_cd4[idx_gone, (t + 1):T_max]  <- NA
    L_vl[idx_gone, (t + 1):T_max]   <- NA
    C_cens[idx_gone, (t + 1):T_max] <- NA
  }
}

# --- Outcome ---
cum_adh   <- rowMeans(A_adh, na.rm = TRUE)
final_vl  <- apply(L_vl, 1, function(x) { v <- x[!is.na(x)]; if (length(v)) tail(v, 1) else NA })
final_cd4 <- apply(L_cd4, 1, function(x) { v <- x[!is.na(x)]; if (length(v)) tail(v, 1) else NA })

lp_y <- -1.0 - 2.0 * cum_adh + 0.5 * final_vl - 0.002 * final_cd4 -
  0.3 * Drug + 0.3 * W_vl_bl
Y <- rbinom(n, 1, plogis(lp_y))
Y[!alive] <- NA

# --- LOCF for censored patients ---
# lmtp handles censoring via the censoring model, but the data itself
# must not contain NA values. Fill post-censoring columns with LOCF.
for (t in 2:T_max) {
  na_cd4 <- is.na(L_cd4[, t])
  na_vl  <- is.na(L_vl[, t])
  na_adh <- is.na(A_adh[, t])
  L_cd4[na_cd4, t] <- L_cd4[na_cd4, t - 1]
  L_vl[na_vl, t]   <- L_vl[na_vl, t - 1]
  A_adh[na_adh, t]  <- A_adh[na_adh, t - 1]
}

# --- Build analysis dataset ---
# Do NOT include an 'id' column; lmtp uses row indices internally.
dat_wide <- tibble(
  Drug = Drug,
  age = W_age, male = W_male, cd4_bl = W_cd4_bl, vl_bl = W_vl_bl
)
for (t in 1:T_max) {
  dat_wide[[paste0("cd4_", t)]]  <- L_cd4[, t]
  dat_wide[[paste0("vl_", t)]]   <- L_vl[, t]
  dat_wide[[paste0("A_", t)]]    <- A_adh[, t]
  dat_wide[[paste0("C_", t)]]    <- C_cens[, t]
}
# Outcome: set censored Y to 0 (arbitrary; censoring model handles it)
dat_wide$Y <- ifelse(is.na(Y), 0, Y)

Note on LMTP API version

This chapter uses lmtp_sdr() throughout, which is the sequentially doubly robust estimator. The V4 simulation code uses lmtp_tmle() instead. Both functions accept identical arguments (data, trt, outcome, baseline, time_vary, cens, shift, mtp, outcome_type, folds, learners_trt, learners_outcome) and target the same estimand. The choice between them is a matter of estimator preference: lmtp_tmle() uses a TMLE-based targeting step, while lmtp_sdr() uses a one-step correction. For most practical purposes, they produce similar results.

All shift functions in this chapter use the correct (data, trt) signature, and lmtp_contrast() calls use the current syntax: lmtp_contrast(fit_shifted, ref = fit_observed, type = "additive"). The mtp = TRUE flag is required whenever the shift function depends on the natural treatment value.

For survival outcomes (not used in this chapter but relevant to the V4 analysis), lmtp returns P(survival), so risk is computed as 1 - theta. Use lmtp::tidy(fit) to extract results as a tidy data frame.

We also estimate the observed (unshifted) outcome as the reference for all contrasts.

23 Approach 1: Marginal Quantile Mapping

23.1 How it works

For each Drug B patient at each visit, we find where their adherence falls in the Drug B empirical CDF and map it to the corresponding quantile of the Drug A empirical CDF. A Drug B patient at the 30th percentile of Drug B adherence gets mapped to the 30th percentile of Drug A adherence. Drug A patients are left unchanged.

Formally, for Drug B patient \(i\) at visit \(t\): \[ d_1(a_{it}) = F_{A|D=A}^{-1}\big(F_{A|D=B}(a_{it})\big) \]

where \(F_{A|D=k}\) is the marginal CDF of adherence among drug \(k\) patients.

This is a rank-preserving transformation: a patient who was relatively compliant among Drug B patients stays relatively compliant, but their absolute adherence level shifts to reflect the Drug A distribution.

23.2 What question it answers

What would happen to Drug B patients’ outcomes if their entire adherence distribution — the shape, spread, and location — were transplanted from the Drug A population?

This captures both the higher mean and any differences in variance or skewness between the two distributions. It asks for a wholesale distributional replacement.

23.3 Implementation

Show code 
# Precompute marginal ECDFs and quantile functions by drug and time
ecdf_B_marginal <- list()
qfun_A_marginal <- list()

for (t in 1:T_max) {
  col <- paste0("A_", t)
  vals_A <- dat_wide[[col]][dat_wide$Drug == 1 & !is.na(dat_wide[[col]])]
  vals_B <- dat_wide[[col]][dat_wide$Drug == 0 & !is.na(dat_wide[[col]])]
  ecdf_B_marginal[[t]] <- ecdf(vals_B)
  # Store sorted Drug A values for quantile lookup
  qfun_A_marginal[[t]] <- sort(vals_A)
}

shift_marginal_qmap <- function(data, trt) {
  t_idx <- as.integer(gsub("A_", "", trt))
  a_obs <- data[[trt]]
  a_shifted <- a_obs

  is_b <- data$Drug == 0 & !is.na(a_obs)
  if (any(is_b)) {
    p_rank <- ecdf_B_marginal[[t_idx]](a_obs[is_b])
    p_rank <- pmax(0.01, pmin(0.99, p_rank))
    sorted_A <- qfun_A_marginal[[t_idx]]
    # Quantile function: index into sorted Drug A values
    a_shifted[is_b] <- sorted_A[pmin(length(sorted_A),
                                      ceiling(p_rank * length(sorted_A)))]
  }
  a_shifted
}

23.4 Interpretation

A negative contrast means the adherence distribution shift reduces virologic failure. The magnitude tells you how much of the Drug A advantage is explained by adherence differences. However, this approach ignores that the adherence gap may differ across patient subgroups — an older patient’s adherence gap between drugs may not be the same as a younger patient’s. Approach 2 addresses this.

24 Approach 2: Conditional Distribution Mapping

24.1 How it works

The marginal quantile map treats all Drug B patients as exchangeable within the Drug B group. But adherence depends on patient characteristics — older patients, those with lower CD4, those with higher viral load may adhere differently. A more principled mapping conditions on covariates, so that the quantile lookup reflects each patient’s clinical profile.

The idea is the same as approach 1 — find a patient’s percentile rank in the Drug B distribution and map it to the corresponding percentile in the Drug A distribution — but now both the CDF and quantile function are conditional on covariates. We estimate these using conditional quantile regression via the quantreg package, fitting separate models at a grid of quantile levels (\(\tau = 0.05, 0.10, \ldots, 0.95\)) for each drug. This avoids the normality assumption that a location-scale model would impose, which matters because adherence is bounded on [0,1] and often skewed.

For a Drug B patient with covariates \(X\) and observed adherence \(a_t\):

  1. Evaluate the fitted Drug B conditional quantiles \(\hat{Q}_{B}(\tau \mid X)\) across the grid and find the \(\tau^*\) where \(a_t\) falls (by interpolation).
  2. Evaluate the Drug A conditional quantile at the same level: \(a_{\text{shifted}} = \hat{Q}_A(\tau^* \mid X)\).

This is a covariate-conditional version of the CDF-inversion used in approach 1.

24.2 What question it answers

What would happen if each Drug B patient’s adherence were set to the level they would most plausibly have achieved had they been on Drug A, given their specific clinical profile?

Two Drug B patients with the same adherence but different covariate profiles may get different shifts, because their covariate-conditional distributions under Drug A differ. A 60-year-old with low CD4 might see a larger shift than a 30-year-old with high CD4, if the Drug A advantage is larger for sicker patients.

24.3 Implementation

Show code 
library(quantreg)

# Grid of quantile levels
tau_grid <- seq(0.05, 0.95, by = 0.05)

# Fit conditional quantile regressions by drug and time point
cqr_models <- list()

for (t in 1:T_max) {
  col <- paste0("A_", t)

  # Covariates available at this time point
  if (t == 1) {
    cov_cols <- c("age", "male", "cd4_bl", "vl_bl")
  } else {
    cov_cols <- c("age", "male", "cd4_bl", "vl_bl",
                  paste0("cd4_", t), paste0("vl_", t),
                  paste0("A_", t - 1))
  }

  fmla <- as.formula(paste(col, "~", paste(cov_cols, collapse = " + ")))

  dat_A <- dat_wide[dat_wide$Drug == 1, ]
  dat_B <- dat_wide[dat_wide$Drug == 0, ]

  # Fit quantile regression at each tau for both drugs
  fits_A <- lapply(tau_grid, function(tau) {
    suppressWarnings(rq(fmla, tau = tau, data = dat_A))
  })
  fits_B <- lapply(tau_grid, function(tau) {
    suppressWarnings(rq(fmla, tau = tau, data = dat_B))
  })

  cqr_models[[t]] <- list(fits_A = fits_A, fits_B = fits_B)
}

# Helper: given a vector of observed values and a matrix of conditional
# quantile predictions (rows = observations, cols = tau grid), find each
# observation's percentile rank by interpolation.
find_tau <- function(a_obs, qmat) {
  n <- length(a_obs)
  tau_out <- numeric(n)
  for (i in seq_len(n)) {
    qs <- qmat[i, ]
    if (a_obs[i] <= qs[1]) {
      tau_out[i] <- tau_grid[1]
    } else if (a_obs[i] >= qs[length(qs)]) {
      tau_out[i] <- tau_grid[length(tau_grid)]
    } else {
      tau_out[i] <- approx(qs, tau_grid, xout = a_obs[i], rule = 2)$y
    }
  }
  tau_out
}

# Helper: given tau values and a matrix of Drug A conditional quantiles,
# return the corresponding adherence values by interpolation.
map_to_A <- function(tau_vals, qmat_A) {
  n <- length(tau_vals)
  a_out <- numeric(n)
  for (i in seq_len(n)) {
    a_out[i] <- approx(tau_grid, qmat_A[i, ], xout = tau_vals[i], rule = 2)$y
  }
  a_out
}

shift_conditional_qmap <- function(data, trt) {
  t_idx <- as.integer(gsub("A_", "", trt))
  a_obs <- data[[trt]]
  a_shifted <- a_obs

  is_b <- data$Drug == 0
  if (any(is_b)) {
    mods <- cqr_models[[t_idx]]

    # Predict conditional quantiles for Drug B patients under both models
    qmat_B <- sapply(mods$fits_B, function(fit) predict(fit, newdata = data[is_b, ]))
    qmat_A <- sapply(mods$fits_A, function(fit) predict(fit, newdata = data[is_b, ]))

    # Ensure quantiles are monotone (fix quantile crossing)
    for (i in seq_len(nrow(qmat_B))) {
      qmat_B[i, ] <- sort(qmat_B[i, ])
      qmat_A[i, ] <- sort(qmat_A[i, ])
    }

    # Step 1: find each Drug B patient's percentile rank in Drug B distribution
    tau_star <- find_tau(a_obs[is_b], qmat_B)

    # Step 2: map to Drug A conditional quantile at that rank
    a_shifted[is_b] <- map_to_A(tau_star, qmat_A)

    # Bound to [0.01, 0.99]
    a_shifted[is_b] <- pmax(0.01, pmin(0.99, a_shifted[is_b]))
  }
  a_shifted
}

24.4 Interpretation

If the conditional and marginal approaches give similar results, the adherence gap between drugs is roughly constant across patient subgroups. If they diverge, the marginal approach is masking heterogeneity — the adherence shift matters more for some patients than others. The conditional approach is generally preferred when covariate-adherence relationships differ meaningfully between drugs, because it avoids shifting patients to implausible adherence levels given their clinical profile.

The quantile regression approach has two advantages over a simpler location-scale normal model. First, it makes no distributional assumption on the residuals — important because adherence is bounded on [0,1] and often skewed. Second, it allows the variance and shape of the conditional adherence distribution to vary with covariates, not just the mean. The cost is computational: we fit \(2 \times T \times |\tau\text{-grid}|\) quantile regressions. Quantile crossing (where predicted quantiles are not monotone) is handled by sorting, which is a standard practical fix.

25 Approach 3: Mean-Calibrated Additive Shift

25.1 How it works

This is the simplest approach. Compute the gap between mean adherence under Drug A and Drug B at each visit, then add that gap to every Drug B patient’s adherence. Drug A patients are left unchanged.

\[ d_3(a_{it}) = \min\big(a_{it} + \delta_t, \; 1\big) \quad \text{where} \quad \delta_t = \bar{A}_t^{D=A} - \bar{A}_t^{D=B} \]

The adherence is capped at 1 because adherence above 100% is meaningless.

25.2 What question it answers

What would happen if we could increase every Drug B patient’s adherence by a uniform amount, so that Drug B’s average adherence matched Drug A’s?

This is a cruder intervention than quantile mapping. It shifts the location of the distribution but preserves the shape (variance, skewness) of Drug B’s adherence distribution. A highly adherent Drug B patient and a poorly adherent one both get the same absolute increase, which may push some patients against the ceiling of 1. The question it answers is closer to “what if an adherence support program raised all Drug B patients’ compliance by a fixed amount?” than to “what if Drug B patients behaved exactly like Drug A patients?”

25.3 Implementation

Show code 
# Compute the mean adherence gap at each visit

      ", Drug B mean =", round(mean_B, 3),

}

The mean-calibrated shift tends to produce slightly different results than the quantile map because it does not reshape the distribution. If the estimate is similar to the quantile-mapping result, the distributional shape difference between drugs is not playing a large role.

This approach is the easiest to explain to a clinical audience: “we added X percentage points to every Drug B patient’s adherence at each visit.” It is also the most conservative, because it does not attempt to match distributional shape.

26 Approach 4: Fixed Additive Shift (Classic LMTP)

26.1 How it works

This is the canonical LMTP modified treatment policy. Instead of calibrating the shift to match Drug A, we apply a fixed, small, uniform increase to every patient’s adherence at every visit, regardless of drug assignment:

\[ d_4(a_{it}) = \min(a_{it} + \delta, \; 1) \]

where \(\delta\) is a prespecified amount (we use \(\delta = 0.10\), i.e., a 10 percentage-point increase).

This is not a cross-drug comparison at all. It is a within-population policy question.

26.2 What question it answers

What would happen if a universal adherence support program (SMS reminders, pill organizers, peer support) increased every patient’s adherence by 10 percentage points?

This is the most directly policy-relevant framing. It does not reference Drug A’s distribution at all. Instead, it asks about the marginal benefit of a small, feasible adherence improvement across the entire cohort. Because the shift is small, it stays close to the observed data, minimizing positivity concerns. This is the intervention most naturally suited to the LMTP framework.

28 What the Results Show

Under observed adherence, the estimated virologic failure rate is about 22%. All four shift interventions reduce it, but by different amounts and with different precision.

The two approaches that most aggressively reshape the Drug B adherence distribution — marginal quantile mapping and the mean-calibrated additive shift — produce the largest reductions, bringing the estimated failure rate down to roughly 5–6% (contrasts of –17 and –16 percentage points, both with 95% CIs excluding zero). These two agree closely despite using very different mechanics, which suggests that in this simulation the adherence gap between drugs is fairly uniform across the distribution and the shape difference between Drug A and Drug B adherence matters less than the location difference.

The conditional distribution mapping — now implemented with conditional quantile regression rather than a normal location-scale model — may produce a different estimate than the marginal approach. Because it conditions on covariates, the shift is more patient-specific: it only moves each Drug B patient’s adherence to what the conditional quantile model predicts they would have achieved under Drug A given their clinical profile. For patients whose covariates predict similar adherence under both drugs, the shift is small. The quantile regression avoids the normality assumption, so the mapping better respects the bounded, potentially skewed shape of real adherence distributions.

The fixed additive shift (+10 percentage points for everyone) yields a –10 percentage point reduction, also with a CI approaching zero. This is a smaller intervention than the cross-drug shifts (the Drug A–Drug B gap is about 23–28 percentage points per visit), so the smaller effect is expected. But it answers a different, more directly actionable question: a realistic adherence support program that raises compliance by 10 points across the board would cut failure rates nearly in half.

The key takeaway is that a substantial portion of the outcome gap between Drug A and Drug B in this simulation is mediated by adherence. The marginal and mean-calibrated approaches suggest that if Drug B patients could adhere at Drug A levels, their failure rates would approach Drug A’s observed rate. The conditional approach is more conservative but still shows a meaningful benefit, and the fixed shift confirms that even a modest, universal adherence improvement would meaningfully reduce virologic failure.

28.1 Connecting shifts to the forgiveness curve

Each of the four shift approaches above gives a different slice of a more fundamental object: the forgiveness curve. The forgiveness curve maps a sustained adherence level to the counterfactual risk of virologic failure:

\[ r_d(a) = P\big(Y = 1 \mid \text{do}(\bar{A} = a), D = d\big) \]

where \(d\) indexes the drug. This curve captures everything about how a drug responds to adherence variation. A “forgiving” drug has a flat curve over a wide PDC range; a less forgiving drug has a steep curve where small adherence drops produce large risk increases.

The four shift approaches relate to this curve as follows:

Fixed additive shift moves each patient along the x-axis of the forgiveness curve by a constant \(\delta\). The LMTP contrast integrates the slope of the curve over the observed adherence distribution, weighted by the shift magnitude. This tells you the marginal benefit of a small adherence improvement, averaged over the patient population. It is a local derivative-like quantity when \(\delta\) is small.

Mean-calibrated additive shift also moves along the x-axis, but by a drug-specific amount calibrated to close the mean adherence gap. It integrates over a larger portion of the curve but still preserves the shape of Drug B’s adherence distribution. The contrast captures how much of the outcome gap between drugs is explained by the location difference in adherence distributions.

Marginal quantile mapping transforms the entire shape of Drug B’s adherence distribution to match Drug A’s. This is not a simple translation along the x-axis — it changes the variance, skewness, and tail behavior. The contrast captures the full distributional difference in adherence, integrating the forgiveness curve under a reshaped distribution.

Conditional quantile mapping does the same distributional transformation, but conditions on patient covariates. This produces a patient-specific shift that respects heterogeneity in adherence-drug relationships.

The forgiveness curve itself can be traced directly using static L-TMLE at a grid of fixed PDC thresholds. The V4 fine-grid LTMLE analysis does this with 13 sliding bins, estimating \(r_d(a)\) at PDC levels from 0.50 to 0.95 in 3–4 percentage-point increments. This traces the curve rather than summarizing it with a single contrast.

Where the curve is well-estimated. The forgiveness curve can only be reliably estimated where there is adequate positivity — enough patients maintaining each PDC level given their covariate history. In the V4 simulation, the curve is well-estimated for PDC \(\geq 0.70\), where both drugs have substantial density. Below 0.70, positivity breaks down because few patients sustain very low adherence for extended periods (patients who disengage tend to be censored). The LMTP shift approaches have an advantage here: because they stay close to the observed distribution, they avoid the low-positivity region where static regimes fail.

The forgiveness curve is thus the unifying object that connects all the estimands in this chapter and in Chapter 3.3. The different shift approaches are different ways of summarizing the same underlying causal surface, chosen based on the scientific question and the positivity constraints of the data.

28.2 Mediation perspective

Informal vs. formal mediation

The decomposition below provides useful intuition for separating adherence-mediated from pharmacological drug effects, but it is an informal controlled direct effect approximation. For a rigorous TMLE-based mediation analysis with formal natural direct and indirect effects (NDE/NIE), identification assumptions, and the crumble package, see the Mediation Analysis chapter.

When comparing two drugs, the total effect on virologic failure has two components:

\[ \underbrace{E[Y(D=A)] - E[Y(D=B)]}_{\text{total effect}} = \underbrace{\text{direct effect}}_{\text{pharmacology}} + \underbrace{\text{indirect effect through adherence}}_{\text{adherence-mediated}} \]

The direct effect captures pharmacological differences between drugs (potency, resistance barrier, half-life) that affect outcomes regardless of adherence. The indirect effect captures the pathway where drug assignment affects adherence (e.g., through pill burden or side effects), which in turn affects outcomes.

The quantile-matching approach provides an informal controlled direct effect (CDE). By matching Drug B’s adherence distribution to Drug A’s, we are holding the adherence pathway constant and asking what outcome difference remains. If the total observed difference between drugs is \(\Delta_{\text{total}}\) and the difference under matched adherence is \(\Delta_{\text{matched}}\), then:

  • \(\Delta_{\text{matched}}\) approximates the direct (pharmacological) effect — the outcome difference when both drugs have the same adherence distribution.
  • \(\Delta_{\text{total}} - \Delta_{\text{matched}}\) approximates the adherence-mediated component — how much of the drug difference operates through the adherence pathway.

This decomposition is informal because a formal causal mediation analysis requires additional assumptions beyond those needed for the shift analysis:

  1. No unmeasured mediator-outcome confounders affected by treatment. In our setting, this means no unmeasured common cause of adherence and failure that is also affected by drug assignment. Time-varying confounders like CD4 and viral load are measured, but there may be unmeasured factors (e.g., motivation, social support) that affect both adherence and outcomes.

  2. Cross-world independence. The formal natural indirect effect requires a “cross-world” counterfactual — the outcome under Drug A with the adherence one would have had under Drug B — which involves quantities from two different interventional worlds simultaneously. This is a stronger assumption than anything required for the shift analysis alone.

Despite these caveats, the shift framework provides valuable intuition about the relative importance of adherence versus pharmacology in explaining drug differences. In the V4 talk companion results, the quantile-matched comparison shows that a substantial portion of Drug A’s observed advantage comes from its better adherence profile, while a residual direct effect remains that reflects genuine pharmacological differences in forgiveness at low adherence levels.

For a formal mediation analysis using stochastic interventions, see Diaz and Hejazi (2020), who develop identification and estimation results for natural direct and indirect effects under stochastic interventions, using the same lmtp infrastructure.

29 Summary: Which Approach for Which Question?

The four approaches are not competing methods for the same estimand. They define different interventions and answer different causal questions.

Approach Intervention Scientific question
Marginal quantile mapping Replace Drug B’s adherence distribution with Drug A’s How much of the outcome gap between drugs is due to the full distributional difference in adherence?
Conditional distribution mapping Replace Drug B’s adherence with what it would have been under Drug A, given each patient’s covariates Same as above, but respecting that the adherence gap may vary across patient profiles
Mean-calibrated additive shift Raise all Drug B patients’ adherence by a uniform amount matching Drug A’s mean What if Drug B’s average adherence could be brought to Drug A’s level via a uniform program?
Fixed additive shift Raise everyone’s adherence by a small fixed amount What is the marginal benefit of a feasible, universal adherence support program?

Approaches 1–3 are cross-drug comparisons: they decompose the Drug A versus Drug B outcome difference into adherence-mediated and pharmacology-mediated components. Approach 4 is a policy evaluation: it estimates the benefit of a concrete intervention without reference to the other drug.

The choice depends on whether the goal is to map the full dose-response relationship (static), evaluate a realistic policy (additive shift), or decompose the drug comparison (quantile match).

For a comparative effectiveness study, approaches 1 or 2 are most appropriate. Approach 2 is preferred when adherence depends strongly on patient characteristics that differ between drug groups. The shift-based approaches (especially 3 and 4) have better positivity properties than static interventions like “always adhere,” because they keep the shifted values close to the observed distribution.

29.1 Check your understanding

Q1. Two of the four shift approaches give very similar estimates. What does that tell you?

When the marginal quantile-mapping and mean-calibrated additive shift give similar results, it means the outcome difference between drugs is driven primarily by the location (mean) of the adherence distributions, not by differences in their shape (variance, skewness, tail behavior). The quantile mapping reshapes the entire distribution while the additive shift only translates it. If both produce similar contrasts, the distributional shape difference between drugs is not playing a large role in explaining outcomes.

Q2. When would you prefer conditional over marginal quantile mapping?

When the adherence gap between drugs varies across patient subgroups. If older patients have a larger Drug A–Drug B adherence gap than younger patients, the marginal approach applies a uniform distributional transformation that may be too aggressive for young patients and too conservative for old ones. The conditional approach uses covariate-specific quantile regressions to produce patient-specific shifts, giving each patient a shift that reflects what they specifically would have achieved under the other drug given their clinical profile.

Q3. The fixed additive shift (+10 pp) is the simplest and most conservative approach. When is it the most useful?

When the question is about a concrete policy intervention rather than a cross-drug comparison. If a health system is considering funding an adherence support program (SMS reminders, pill organizers, peer counselors) expected to raise adherence by about 10 percentage points, the fixed shift directly estimates the expected outcome improvement. It answers “what would this program achieve?” without reference to any other drug, making it the most directly actionable for policy planning. It also has the strongest positivity properties because the shift is small and uniform.

# Overview

Chapter 3.3 introduced a distribution-shift intervention using quantile mapping to compare two antiretroviral drugs under a common adherence distribution. That approach is one principled way to operationalize the question "what if Drug B patients adhered like Drug A patients?" But it is not the only way. Different shift functions encode different assumptions about what "adhering like Drug A" means, and they answer subtly different causal questions.

This supplement implements four approaches on the same simulated data:

1. **Marginal quantile mapping** --- map Drug B adherence to the Drug A distribution via the empirical CDF.
2. **Conditional distribution mapping** --- same idea, but condition on baseline covariates so the mapping respects patient characteristics.
3. **Mean-calibrated additive shift** --- add a constant to Drug B adherence so its mean matches Drug A's.
4. **Fixed additive shift** --- increase every patient's adherence by a small fixed amount, regardless of drug.

Each approach defines a different modified treatment policy for `lmtp`. We run all four, compare estimates, and explain what scientific question each one answers.

```{r, cache=TRUE}
shift_identity <- function(data, trt) data[[trt]]
```

# Approach 1: Marginal Quantile Mapping

## How it works

For each Drug B patient at each visit, we find where their adherence falls in the Drug B empirical CDF and map it to the corresponding quantile of the Drug A empirical CDF. A Drug B patient at the 30th percentile of Drug B adherence gets mapped to the 30th percentile of Drug A adherence. Drug A patients are left unchanged.

Formally, for Drug B patient $i$ at visit $t$:
$$
d_1(a_{it}) = F_{A|D=A}^{-1}\big(F_{A|D=B}(a_{it})\big)
$$

where $F_{A|D=k}$ is the marginal CDF of adherence among drug $k$ patients.

This is a rank-preserving transformation: a patient who was relatively compliant among Drug B patients stays relatively compliant, but their absolute adherence level shifts to reflect the Drug A distribution.

## What question it answers

*What would happen to Drug B patients' outcomes if their entire adherence distribution --- the shape, spread, and location --- were transplanted from the Drug A population?*

This captures both the higher mean and any differences in variance or skewness between the two distributions. It asks for a wholesale distributional replacement.

## Implementation

```{r}
# Precompute marginal ECDFs and quantile functions by drug and time
ecdf_B_marginal <- list()
qfun_A_marginal <- list()

for (t in 1:T_max) {
  col <- paste0("A_", t)
  vals_A <- dat_wide[[col]][dat_wide$Drug == 1 & !is.na(dat_wide[[col]])]
  vals_B <- dat_wide[[col]][dat_wide$Drug == 0 & !is.na(dat_wide[[col]])]
  ecdf_B_marginal[[t]] <- ecdf(vals_B)
  # Store sorted Drug A values for quantile lookup
  qfun_A_marginal[[t]] <- sort(vals_A)
}

shift_marginal_qmap <- function(data, trt) {
  t_idx <- as.integer(gsub("A_", "", trt))
  a_obs <- data[[trt]]
  a_shifted <- a_obs

  is_b <- data$Drug == 0 & !is.na(a_obs)
  if (any(is_b)) {
    p_rank <- ecdf_B_marginal[[t_idx]](a_obs[is_b])
    p_rank <- pmax(0.01, pmin(0.99, p_rank))
    sorted_A <- qfun_A_marginal[[t_idx]]
    # Quantile function: index into sorted Drug A values
    a_shifted[is_b] <- sorted_A[pmin(length(sorted_A),
                                      ceiling(p_rank * length(sorted_A)))]
  }
  a_shifted
}
```

# Approach 2: Conditional Distribution Mapping

## How it works

The marginal quantile map treats all Drug B patients as exchangeable within the Drug B group. But adherence depends on patient characteristics --- older patients, those with lower CD4, those with higher viral load may adhere differently. A more principled mapping conditions on covariates, so that the quantile lookup reflects each patient's clinical profile.

The idea is the same as approach 1 --- find a patient's percentile rank in the Drug B distribution and map it to the corresponding percentile in the Drug A distribution --- but now both the CDF and quantile function are conditional on covariates. We estimate these using **conditional quantile regression** via the `quantreg` package, fitting separate models at a grid of quantile levels ($\tau = 0.05, 0.10, \ldots, 0.95$) for each drug. This avoids the normality assumption that a location-scale model would impose, which matters because adherence is bounded on [0,1] and often skewed.

For a Drug B patient with covariates $X$ and observed adherence $a_t$:

1. Evaluate the fitted Drug B conditional quantiles $\hat{Q}_{B}(\tau \mid X)$ across the grid and find the $\tau^*$ where $a_t$ falls (by interpolation).
2. Evaluate the Drug A conditional quantile at the same level: $a_{\text{shifted}} = \hat{Q}_A(\tau^* \mid X)$.

This is a covariate-conditional version of the CDF-inversion used in approach 1.

## What question it answers

*What would happen if each Drug B patient's adherence were set to the level they would most plausibly have achieved had they been on Drug A, given their specific clinical profile?*

Two Drug B patients with the same adherence but different covariate profiles may get different shifts, because their covariate-conditional distributions under Drug A differ. A 60-year-old with low CD4 might see a larger shift than a 30-year-old with high CD4, if the Drug A advantage is larger for sicker patients.

## Implementation

```{r}
library(quantreg)

# Grid of quantile levels
tau_grid <- seq(0.05, 0.95, by = 0.05)

# Fit conditional quantile regressions by drug and time point
cqr_models <- list()

for (t in 1:T_max) {
  col <- paste0("A_", t)

  # Covariates available at this time point
  if (t == 1) {
    cov_cols <- c("age", "male", "cd4_bl", "vl_bl")
  } else {
    cov_cols <- c("age", "male", "cd4_bl", "vl_bl",
                  paste0("cd4_", t), paste0("vl_", t),
                  paste0("A_", t - 1))
  }

  fmla <- as.formula(paste(col, "~", paste(cov_cols, collapse = " + ")))

  dat_A <- dat_wide[dat_wide$Drug == 1, ]
  dat_B <- dat_wide[dat_wide$Drug == 0, ]

  # Fit quantile regression at each tau for both drugs
  fits_A <- lapply(tau_grid, function(tau) {
    suppressWarnings(rq(fmla, tau = tau, data = dat_A))
  })
  fits_B <- lapply(tau_grid, function(tau) {
    suppressWarnings(rq(fmla, tau = tau, data = dat_B))
  })

  cqr_models[[t]] <- list(fits_A = fits_A, fits_B = fits_B)
}

# Helper: given a vector of observed values and a matrix of conditional
# quantile predictions (rows = observations, cols = tau grid), find each
# observation's percentile rank by interpolation.
find_tau <- function(a_obs, qmat) {
  n <- length(a_obs)
  tau_out <- numeric(n)
  for (i in seq_len(n)) {
    qs <- qmat[i, ]
    if (a_obs[i] <= qs[1]) {
      tau_out[i] <- tau_grid[1]
    } else if (a_obs[i] >= qs[length(qs)]) {
      tau_out[i] <- tau_grid[length(tau_grid)]
    } else {
      tau_out[i] <- approx(qs, tau_grid, xout = a_obs[i], rule = 2)$y
    }
  }
  tau_out
}

# Helper: given tau values and a matrix of Drug A conditional quantiles,
# return the corresponding adherence values by interpolation.
map_to_A <- function(tau_vals, qmat_A) {
  n <- length(tau_vals)
  a_out <- numeric(n)
  for (i in seq_len(n)) {
    a_out[i] <- approx(tau_grid, qmat_A[i, ], xout = tau_vals[i], rule = 2)$y
  }
  a_out
}

shift_conditional_qmap <- function(data, trt) {
  t_idx <- as.integer(gsub("A_", "", trt))
  a_obs <- data[[trt]]
  a_shifted <- a_obs

  is_b <- data$Drug == 0
  if (any(is_b)) {
    mods <- cqr_models[[t_idx]]

    # Predict conditional quantiles for Drug B patients under both models
    qmat_B <- sapply(mods$fits_B, function(fit) predict(fit, newdata = data[is_b, ]))
    qmat_A <- sapply(mods$fits_A, function(fit) predict(fit, newdata = data[is_b, ]))

    # Ensure quantiles are monotone (fix quantile crossing)
    for (i in seq_len(nrow(qmat_B))) {
      qmat_B[i, ] <- sort(qmat_B[i, ])
      qmat_A[i, ] <- sort(qmat_A[i, ])
    }

    # Step 1: find each Drug B patient's percentile rank in Drug B distribution
    tau_star <- find_tau(a_obs[is_b], qmat_B)

    # Step 2: map to Drug A conditional quantile at that rank
    a_shifted[is_b] <- map_to_A(tau_star, qmat_A)

    # Bound to [0.01, 0.99]
    a_shifted[is_b] <- pmax(0.01, pmin(0.99, a_shifted[is_b]))
  }
  a_shifted
}
```

## Interpretation

If the conditional and marginal approaches give similar results, the adherence gap between drugs is roughly constant across patient subgroups. If they diverge, the marginal approach is masking heterogeneity --- the adherence shift matters more for some patients than others. The conditional approach is generally preferred when covariate-adherence relationships differ meaningfully between drugs, because it avoids shifting patients to implausible adherence levels given their clinical profile.

The quantile regression approach has two advantages over a simpler location-scale normal model. First, it makes no distributional assumption on the residuals --- important because adherence is bounded on [0,1] and often skewed. Second, it allows the variance and shape of the conditional adherence distribution to vary with covariates, not just the mean. The cost is computational: we fit $2 \times T \times |\tau\text{-grid}|$ quantile regressions. Quantile crossing (where predicted quantiles are not monotone) is handled by sorting, which is a standard practical fix.

# Approach 3: Mean-Calibrated Additive Shift

## How it works

This is the simplest approach. Compute the gap between mean adherence under Drug A and Drug B at each visit, then add that gap to every Drug B patient's adherence. Drug A patients are left unchanged.

$$
d_3(a_{it}) = \min\big(a_{it} + \delta_t, \; 1\big) \quad \text{where} \quad \delta_t = \bar{A}_t^{D=A} - \bar{A}_t^{D=B}
$$

The adherence is capped at 1 because adherence above 100% is meaningless.

## What question it answers

*What would happen if we could increase every Drug B patient's adherence by a uniform amount, so that Drug B's average adherence matched Drug A's?*

This is a cruder intervention than quantile mapping. It shifts the location of the distribution but preserves the shape (variance, skewness) of Drug B's adherence distribution. A highly adherent Drug B patient and a poorly adherent one both get the same absolute increase, which may push some patients against the ceiling of 1. The question it answers is closer to "what if an adherence support program raised all Drug B patients' compliance by a fixed amount?" than to "what if Drug B patients behaved exactly like Drug A patients?"

## Implementation

```{r}
# Compute the mean adherence gap at each visit
adh_gaps <- numeric(T_max)
for (t in 1:T_max) {
  col <- paste0("A_", t)
  mean_A <- mean(dat_wide[[col]][dat_wide$Drug == 1], na.rm = TRUE)
  mean_B <- mean(dat_wide[[col]][dat_wide$Drug == 0], na.rm = TRUE)

  cat("Visit", t, ": Drug A mean =", round(mean_A, 3),
      ", Drug B mean =", round(mean_B, 3),
      ", gap =", round(adh_gaps[t], 3), "\n")
}
```

The mean-calibrated shift tends to produce slightly different results than the quantile map because it does not reshape the distribution. If Drug B's adherence distribution has heavier left tail than Drug A's (more patients with very low adherence), the additive shift pushes those patients up but does not compress the tail the way quantile mapping would. Conversely, high-adherence Drug B patients may hit the ceiling at 1, wasting some of the shift. If the estimate is similar to the quantile-mapping result, the distributional shape difference between drugs is not playing a large role.

This approach is the easiest to explain to a clinical audience: "we added X percentage points to every Drug B patient's adherence at each visit." It is also the most conservative, because it does not attempt to match distributional shape.

# Approach 4: Fixed Additive Shift (Classic LMTP)

## How it works

This is the canonical LMTP modified treatment policy. Instead of calibrating the shift to match Drug A, we apply a fixed, small, uniform increase to **every** patient's adherence at every visit, regardless of drug assignment:

$$
d_4(a_{it}) = \min(a_{it} + \delta, \; 1)
$$

where $\delta$ is a prespecified amount (we use $\delta = 0.10$, i.e., a 10 percentage-point increase).

This is not a cross-drug comparison at all. It is a within-population policy question.

## What question it answers

*What would happen if a universal adherence support program (SMS reminders, pill organizers, peer support) increased every patient's adherence by 10 percentage points?*

This is the most directly policy-relevant framing. It does not reference Drug A's distribution at all. Instead, it asks about the marginal benefit of a small, feasible adherence improvement across the entire cohort. Because the shift is small, it stays close to the observed data, minimizing positivity concerns. This is the intervention most naturally suited to the LMTP framework.

## Implementation

Under observed adherence, the estimated virologic failure rate is about 22%. All four shift interventions reduce it, but by different amounts and with different precision.

The two approaches that most aggressively reshape the Drug B adherence distribution --- marginal quantile mapping and the mean-calibrated additive shift --- produce the largest reductions, bringing the estimated failure rate down to roughly 5--6% (contrasts of --17 and --16 percentage points, both with 95% CIs excluding zero). These two agree closely despite using very different mechanics, which suggests that in this simulation the adherence gap between drugs is fairly uniform across the distribution and the shape difference between Drug A and Drug B adherence matters less than the location difference.

The conditional distribution mapping --- now implemented with conditional quantile regression rather than a normal location-scale model --- may produce a different estimate than the marginal approach. Because it conditions on covariates, the shift is more patient-specific: it only moves each Drug B patient's adherence to what the conditional quantile model predicts they would have achieved under Drug A given their clinical profile. For patients whose covariates predict similar adherence under both drugs, the shift is small. The quantile regression avoids the normality assumption, so the mapping better respects the bounded, potentially skewed shape of real adherence distributions.

The fixed additive shift (+10 percentage points for everyone) yields a --10 percentage point reduction, also with a CI approaching zero. This is a smaller intervention than the cross-drug shifts (the Drug A--Drug B gap is about 23--28 percentage points per visit), so the smaller effect is expected. But it answers a different, more directly actionable question: a realistic adherence support program that raises compliance by 10 points across the board would cut failure rates nearly in half.

The key takeaway is that a substantial portion of the outcome gap between Drug A and Drug B in this simulation is mediated by adherence. The marginal and mean-calibrated approaches suggest that if Drug B patients could adhere at Drug A levels, their failure rates would approach Drug A's observed rate. The conditional approach is more conservative but still shows a meaningful benefit, and the fixed shift confirms that even a modest, universal adherence improvement would meaningfully reduce virologic failure.


## Connecting shifts to the forgiveness curve

Each of the four shift approaches above gives a different slice of a more fundamental object: the **forgiveness curve**. The forgiveness curve maps a sustained adherence level to the counterfactual risk of virologic failure:

$$
r_d(a) = P\big(Y = 1 \mid \text{do}(\bar{A} = a), D = d\big)
$$

where $d$ indexes the drug. This curve captures everything about how a drug responds to adherence variation. A "forgiving" drug has a flat curve over a wide PDC range; a less forgiving drug has a steep curve where small adherence drops produce large risk increases.

The four shift approaches relate to this curve as follows:

**Fixed additive shift** moves each patient along the x-axis of the forgiveness curve by a constant $\delta$. The LMTP contrast integrates the slope of the curve over the observed adherence distribution, weighted by the shift magnitude. This tells you the marginal benefit of a small adherence improvement, averaged over the patient population. It is a local derivative-like quantity when $\delta$ is small.

**Mean-calibrated additive shift** also moves along the x-axis, but by a drug-specific amount calibrated to close the mean adherence gap. It integrates over a larger portion of the curve but still preserves the shape of Drug B's adherence distribution. The contrast captures how much of the outcome gap between drugs is explained by the *location* difference in adherence distributions.

**Marginal quantile mapping** transforms the entire shape of Drug B's adherence distribution to match Drug A's. This is not a simple translation along the x-axis --- it changes the variance, skewness, and tail behavior. The contrast captures the full distributional difference in adherence, integrating the forgiveness curve under a reshaped distribution.

**Conditional quantile mapping** does the same distributional transformation, but conditions on patient covariates. This produces a patient-specific shift that respects heterogeneity in adherence-drug relationships.

The forgiveness curve itself can be traced directly using **static L-TMLE** at a grid of fixed PDC thresholds. The V4 fine-grid LTMLE analysis does this with 13 sliding bins, estimating $r_d(a)$ at PDC levels from 0.50 to 0.95 in 3--4 percentage-point increments. This traces the curve rather than summarizing it with a single contrast.

**Where the curve is well-estimated.** The forgiveness curve can only be reliably estimated where there is adequate positivity --- enough patients maintaining each PDC level given their covariate history. In the V4 simulation, the curve is well-estimated for PDC $\geq 0.70$, where both drugs have substantial density. Below 0.70, positivity breaks down because few patients sustain very low adherence for extended periods (patients who disengage tend to be censored). The LMTP shift approaches have an advantage here: because they stay close to the observed distribution, they avoid the low-positivity region where static regimes fail.

The forgiveness curve is thus the unifying object that connects all the estimands in this chapter and in Chapter 3.3. The different shift approaches are different ways of summarizing the same underlying causal surface, chosen based on the scientific question and the positivity constraints of the data.


## Mediation perspective

::: {.callout-note}
## Informal vs. formal mediation
The decomposition below provides useful intuition for separating adherence-mediated from pharmacological drug effects, but it is an *informal* controlled direct effect approximation. For a rigorous TMLE-based mediation analysis with formal natural direct and indirect effects (NDE/NIE), identification assumptions, and the `crumble` package, see the [Mediation Analysis chapter](03-10-tmle-mediation.qmd).
:::

When comparing two drugs, the total effect on virologic failure has two components:

$$
\underbrace{E[Y(D=A)] - E[Y(D=B)]}_{\text{total effect}} = \underbrace{\text{direct effect}}_{\text{pharmacology}} + \underbrace{\text{indirect effect through adherence}}_{\text{adherence-mediated}}
$$

The direct effect captures pharmacological differences between drugs (potency, resistance barrier, half-life) that affect outcomes regardless of adherence. The indirect effect captures the pathway where drug assignment affects adherence (e.g., through pill burden or side effects), which in turn affects outcomes.

The quantile-matching approach provides an informal controlled direct effect (CDE). By matching Drug B's adherence distribution to Drug A's, we are holding the adherence pathway constant and asking what outcome difference remains. If the total observed difference between drugs is $\Delta_{\text{total}}$ and the difference under matched adherence is $\Delta_{\text{matched}}$, then:

- $\Delta_{\text{matched}}$ approximates the direct (pharmacological) effect --- the outcome difference when both drugs have the same adherence distribution.
- $\Delta_{\text{total}} - \Delta_{\text{matched}}$ approximates the adherence-mediated component --- how much of the drug difference operates through the adherence pathway.

This decomposition is *informal* because a formal causal mediation analysis requires additional assumptions beyond those needed for the shift analysis:

1. **No unmeasured mediator-outcome confounders** affected by treatment. In our setting, this means no unmeasured common cause of adherence and failure that is also affected by drug assignment. Time-varying confounders like CD4 and viral load are measured, but there may be unmeasured factors (e.g., motivation, social support) that affect both adherence and outcomes.

2. **Cross-world independence.** The formal natural indirect effect requires a "cross-world" counterfactual --- the outcome under Drug A with the adherence one would have had under Drug B --- which involves quantities from two different interventional worlds simultaneously. This is a stronger assumption than anything required for the shift analysis alone.

Despite these caveats, the shift framework provides valuable intuition about the relative importance of adherence versus pharmacology in explaining drug differences. In the V4 talk companion results, the quantile-matched comparison shows that a substantial portion of Drug A's observed advantage comes from its better adherence profile, while a residual direct effect remains that reflects genuine pharmacological differences in forgiveness at low adherence levels.

For a formal mediation analysis using stochastic interventions, see Diaz and Hejazi (2020), who develop identification and estimation results for natural direct and indirect effects under stochastic interventions, using the same `lmtp` infrastructure.


# Summary: Which Approach for Which Question?

The four approaches are not competing methods for the same estimand. They define different interventions and answer different causal questions.

| Approach | Intervention | Scientific question |
|----------|-------------|-------------------|
| Marginal quantile mapping | Replace Drug B's adherence distribution with Drug A's | How much of the outcome gap between drugs is due to the full distributional difference in adherence? |
| Conditional distribution mapping | Replace Drug B's adherence with what it would have been under Drug A, given each patient's covariates | Same as above, but respecting that the adherence gap may vary across patient profiles |
| Mean-calibrated additive shift | Raise all Drug B patients' adherence by a uniform amount matching Drug A's mean | What if Drug B's average adherence could be brought to Drug A's level via a uniform program? |
| Fixed additive shift | Raise everyone's adherence by a small fixed amount | What is the marginal benefit of a feasible, universal adherence support program? |

Approaches 1--3 are cross-drug comparisons: they decompose the Drug A versus Drug B outcome difference into adherence-mediated and pharmacology-mediated components. Approach 4 is a policy evaluation: it estimates the benefit of a concrete intervention without reference to the other drug.

### V4 simulation context: static versus shift approaches

The V4 Monte Carlo simulation provides a broader perspective by comparing static and shift-based approaches on the same DGP. The table below summarizes the three main estimation strategies and their properties in the simulation context.

| Approach | Estimand | Scientific question | Positivity | V4 MC finding |
|----------|----------|-------------------|------------|--------------|
| Static regime (L-TMLE) | $E[Y(\bar{A} \geq a)]$ at grid of $a$ | Traces the full forgiveness curve: what is risk at each sustained PDC level? | Requires patients at each fixed PDC level; breaks down below PDC $\approx 0.70$ | Well-calibrated coverage at high PDC; bias and variance increase at low PDC where positivity fails |
| Additive shift (LMTP) | $E[Y(\bar{A} + \delta)]$ | Policy-relevant: what if adherence improved by $\delta$ (e.g., 10 pp)? | Stays within support as long as shift is small; best positivity properties | Low bias ($< 0.01$) under strong time-varying confounding when shift stays within support |
| Quantile match (LMTP) | $E[Y(\bar{A}^{d_{\text{ref}}})]$ | Isolates pharmacological effect: what if Drug A had Drug B's adherence distribution? | Shifted values are observed values from the reference drug; within-support by construction | Provides informal decomposition of total drug effect into adherence-mediated and direct components |

All three approaches are functions of the same counterfactual risk surface $P(Y \mid \text{do}(\bar{A}), D)$. The static regime evaluates this surface at fixed points; the additive shift integrates it under a translated distribution; and the quantile match integrates it under a reshaped distribution. The choice depends on whether the goal is to map the full dose-response relationship (static), evaluate a realistic policy (additive shift), or decompose the drug comparison (quantile match).

For a comparative effectiveness study, approaches 1 or 2 are most appropriate. Approach 2 is preferred when adherence depends strongly on patient characteristics that differ between drug groups. For health policy planning, approach 4 is the most directly useful. Approach 3 is a middle ground --- a cross-drug comparison using the simplest possible shift function.

In all cases, the validity of the estimate depends on the same identification assumptions: consistency, sequential exchangeability, and positivity. The shift-based approaches (especially 3 and 4) have better positivity properties than static interventions like "always adhere," because they keep the shifted values close to the observed distribution.

<summary><strong>Q1. Two of the four shift approaches give very similar estimates. What does that tell you?</strong></summary>

When the marginal quantile-mapping and mean-calibrated additive shift give similar results, it means the outcome difference between drugs is driven primarily by the *location* (mean) of the adherence distributions, not by differences in their *shape* (variance, skewness, tail behavior). The quantile mapping reshapes the entire distribution while the additive shift only translates it. If both produce similar contrasts, the distributional shape difference between drugs is not playing a large role in explaining outcomes.

</details>

<details>
<summary><strong>Q2. When would you prefer conditional over marginal quantile mapping?</strong></summary>

When the adherence gap between drugs varies across patient subgroups. If older patients have a larger Drug A--Drug B adherence gap than younger patients, the marginal approach applies a uniform distributional transformation that may be too aggressive for young patients and too conservative for old ones. The conditional approach uses covariate-specific quantile regressions to produce patient-specific shifts, giving each patient a shift that reflects what they specifically would have achieved under the other drug given their clinical profile.

</details>

<details>
<summary><strong>Q3. The fixed additive shift (+10 pp) is the simplest and most conservative approach. When is it the most useful?</strong></summary>

When the question is about a concrete policy intervention rather than a cross-drug comparison. If a health system is considering funding an adherence support program (SMS reminders, pill organizers, peer counselors) expected to raise adherence by about 10 percentage points, the fixed shift directly estimates the expected outcome improvement. It answers "what would this program achieve?" without reference to any other drug, making it the most directly actionable for policy planning. It also has the strongest positivity properties because the shift is small and uniform.

</details>
